I am seeing a dream of creating an App which can encode any password, message or secret chat and can be decoded by only you. You dont need to remember every password but you should have encrypted code somewhere(which is anyways not readable).

The main motive behind creating this app is to create encoded message of your code with your given key. This is an Open source project and feel free to add your ideas. This key is X digit number which can be used to encode as well as to decode.
